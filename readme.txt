Сервис хранения номеров телефона в зашифрованном виде с доступом по e-mail.

1. Конфигурация для доступа к базе задается в /config/main.php в разделе db
2. Для создания содержимого базы нужно выполнить файл /storage.sql
3. Само приложение находится в /application/storage.
4. Принцип работы и нюансы приложения storage:
 - в базе, в табличке phone храним зашифрованный с ключем алгоритмом blowfish номер телефона, а так же SHA1-хэш e-mail'а;
 - ключ для шифрования - e-mail;
 - при поиске номера по базе - ищем по хэшу e-mail'а;
 - поле хэша уникально, поэтому приложение будет информировать об уже имеющемся в базе e-mail и не давать повторной записи с таким же e-mail;
